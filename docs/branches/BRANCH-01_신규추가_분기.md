# BRANCH-01: 신규 / 추가 분기

> 분기 발생 시점: **0001 사업자번호 입력** 화면에서 조회 결과에 따라 분기

---

## 분기 조건

| 조건 | 다음 화면 | 설명 |
|------|----------|------|
| 사업자번호 조회 결과 → 기존 회원 **없음** | → `0002` 신규 대상 통합 발급 안내 | **신규 발급** |
| 사업자번호 조회 결과 → 기존 회원 **있음** | → `0003` 추가 대상 통합 발급 안내 | **추가 발급** |

## 처리계 연동

```
사업자번호 입력 → 회원원장DB 조회
  ├─ 신용카드 발급 제한 여부 확인 (심사)
  ├─ 최근 1개월 내 동일 사유 반송건 2회 이상 → 접수 거절
  └─ 추가발급 선택 시: 기존 회원 원장 정보 신청 페이지 자동 반영
```

## 영향 범위

| 시나리오 | 신규 흐름 | 추가 흐름 |
|----------|----------|---------|
| 법사자_신용 | 0002→...→0034 | 0003→...→0052 |
| 법사자_체크 | 0002→...→0034 | 0003→...→0052 |
| 개사자_신용 | 0002→0053→...→0065 | 0003→0053→...→0065 |
| 개사자_체크 | 0002→0066→...→0074 | (미확보) |
