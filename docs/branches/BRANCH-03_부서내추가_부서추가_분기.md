# BRANCH-03: 부서내추가발급 / 부서추가 분기

> 분기 발생 시점: **추가발급 + 공용카드** 조합에서만 발생  
> **해당 시나리오: 법사자_신용(추가_공용), 법사자_체크(추가_공용)만 해당**

---

## 분기 발생 조건

```
신규/추가 분기 → [추가] 선택
  + 공용/개별 분기 → [공용] 선택
    → 0043 발급 대상 부서 팝업에서 분기 발생
```

## 분기 조건

| 선택 | 다음 화면 | 설명 |
|------|----------|------|
| **부서내 추가발급** | → 기존 부서 내 추가 발급 흐름 | 기존 부서에 카드 추가 |
| **부서 추가** | → 새 부서 생성 후 발급 흐름 | 신규 부서 생성 필요 |

## 부서내추가 vs 부서추가 차이

| 항목 | 부서내추가발급 | 부서추가 |
|------|--------------|--------|
| 부서명 | 기존 부서 선택 | 신규 부서명 **직접 입력** |
| 추가 화면 | 없음 | `0047` 대표자_사업자정보, `0048` 업체(부서)정보 |
| 한도 입력 | `0050` 부서별한도 (공통) | `0050` 부서별한도 |
| 완료 | `0052` 신청완료_카드신청정보(확인용) | `0052` 신청완료_카드신청정보(확인용) |

## 부서내추가 흐름 (`SC-01-C` 요약)

```
...→ 0043 발급대상부서 팝업 (부서내추가 선택)
→ 0010 신청인정보
→ 0011 신용정보 조회 동의
→ 0012 신청인 본인인증
→ 0013 신청인 CDD 조건부 검증
→ 0018 필수확인사항
→ 0045 기본정보
→ 0021 카드선택 → 0024 신청매수 → 0025 발급확인 → 0026 카드수령지
→ 0042 법인 EDD정보
→ 0027 추천인 → 0028 공동인증서 인증
→ 0029 카드정보(확인용) → 0051 업체(부서)정보(확인용)
→ 0032 법인인증
→ 0052 신청완료
```

## 부서추가 흐름 (`SC-01-D` 요약)

```
...→ 0043 발급대상부서 팝업 (부서추가 선택)
→ 0047 대표자_사업자정보  ← 부서추가 전용
→ 0048 업체(부서)정보      ← 부서추가 전용
→ 0049 기본정보2
→ 0050 부서별한도(요청한도 입력)  ← 부서추가 전용
→ 0020 명세서 받으실 곳
→ 0021 카드선택 → 0022 결제일 → 0023 결제계좌 → 0024 신청매수
→ 0025 발급확인 → 0026 카드수령지
→ 0042 법인 EDD정보
→ 0027 추천인 → 0028 공동인증서 인증
→ 0029 카드정보(확인용) → 0051 업체(부서)정보(확인용)
→ 0032 법인인증
→ 0052 신청완료
```
