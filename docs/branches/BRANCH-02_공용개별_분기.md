# BRANCH-02: 공용카드 / 개별카드 분기

> 분기 발생 시점: **0008 법인 공동인증서(기업용)이용안내** 화면에서 고객 선택에 따라 분기  
> **해당 시나리오: 법사자_신용, 법사자_체크만 해당** (개사자는 분기 없음)

---

## 분기 조건

| 선택 | 다음 화면 | 설명 |
|------|----------|------|
| **공용카드** 선택 | → `0009` 약관동의 (신용) 또는 체크카드 약관동의 | 기업 명의 공용카드 |
| **개별카드** 선택 | → `0035` 개별 카드 회원 동의서 작성안내 | 개인 명의 개별카드 |

## 공용카드 vs 개별카드 차이

| 항목 | 공용카드 | 개별카드 |
|------|---------|---------|
| 카드 소유 | 기업 | 개인 (카드 사용자) |
| 약관동의 | `0009` 신용/체크카드 상품 약관 | `0035` 개별 카드 회원 동의서 |
| 계좌 등록 | `0023` 결제계좌 (기업계좌) | `0036` 개인명의계좌 등록 |
| 신청매수 | `0024` 신청매수 입력 | 해당 없음 (1매 고정) |
| 개별카드 대상자 | 해당 없음 | `0037` 개별카드 발급 대상 고객 정보 |
| 신용정보 동의 | `0011` 신청자 동의 | `0039` 개인신용정보 동의서 |

## 개별카드 전용 흐름

```
0035 개별카드 회원 동의서 작성 안내
→ 0036 개별카드 사용자 개인명의계좌 등록
→ 0037 개별카드 발급 대상 고객 정보 (LMS 발송)
→ 0038 개별카드 동의 조회(조회하기) ← 고객이 LMS 동의 완료 후 새로고침
→ 0039 개인신용정보 동의서
→ 0040 개별카드 동의 조회(신청하기)
→ [공통 흐름 합류: 0010 신청인정보부터]
```
